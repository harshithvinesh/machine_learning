Open terminal/command prompt and navigate to your project folder (e.g., cd ~/Desktop/my-mern-project).

If not already a Git repo: git init, then add remote: git remote add origin https://github.com/yourusername/your-repo.git (create repo on GitHub first).

Place your file in the folder (e.g., copy server.js).

Stage changes: git add filename.js (or git add . for all files).

Commit: git commit -m "Add backend server file".

Push to GitHub: git push -u origin main (use master if that's your default branch; enter credentials/PAT if prompted).

Verify on GitHub.com – refresh to see the file.
